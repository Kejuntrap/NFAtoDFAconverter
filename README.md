# Taoyuanxiang.java

NFAをDFAに書き下す

フォーマット

使う文字の種類 文字の列挙(スペース区切り)
状態数 遷移数
頂点名 受理状態かどうか 開始地点かどうか

︙

遷移元の状態名 文字 遷移先の状態名

︙

当然(?)ですが、DFAっぽいNFAほど状態数が多いので簡略化が効果を発揮します。
